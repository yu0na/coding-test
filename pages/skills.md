# 기술 스택

- <strong>React</strong> 또는 <strong>Next.js</strong> 사용은 필수입니다.

- <strong>TypeScript</strong> 사용은 필수입니다.

- <strong>상태 관리 라이브러리</strong>를 반드시 사용해 주세요.
(예: TanStack Query, Zustand, Recoil, Jotai 등)

- <strong>테이블 UI는 직접 구현해야 하며, 테이블 관련 UI 라이브러리는 사용할 수 없습니다.</strong>

- 스타일링 방식은 자유롭게 선택할 수 있습니다.
(예: Tailwind CSS, styled-components, emotion 등)
